This section describes how Nebula Commander is built in CI and how to build everything yourself.

- **[GitHub Actions](/docs/building-and-ci/github-actions/)** – What the workflows do, when they run (tags, pull requests, manual), and where artifacts and images are published.
- **[Manual builds](/docs/building-and-ci/manual-builds/)** – How to build ncclient binaries (all platforms), the Windows tray app, the Windows MSI, and Docker images locally without CI.

Use these docs to release new versions, reproduce CI builds, or customize the pipelines.
